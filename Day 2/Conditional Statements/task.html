Basic If Statements

Let's introduce you to conditional statements! Conditional statements are essentially how your program can make choices. It is normally in the form of an if statement, but it can be found in other forms too! Let's explore if statements today.

Think about it like real life. If you need a haircut, you go to the barber, but if you need groceries, you go to the supermarket. That's essentially how conditional statements work in Java on a basic level. In fact, they can do more than just check if something is true or false. In the above example, needing a haircut or needing groceries are represented by booleans, since they can only be true or false. Java conditional statements can check if objects are (The corresponding symbols are beside them):

  • Less than - <
  • Less than or equal to - <=
  • Greater than - >
  • Greater than or equal to - >=/li>
  • Equal to - ==
  • Not equal to - !

Let's try it out by making a simple if statement. For this example, let's create the example above. Make a boolean representing if you need groceries or not in the main method. Then, use the following syntax: if(condition){then}

Using the above syntax, just enter the name of the boolean as the condition, and a print statement that says "I will go to the store" as the then

In this case, you should have:

   
    public class ConditionalStatementPractice{
        public static void main(String[] args){
            boolean groceries = true; // or false if you would like
            if(groceries){
                System.out.println("I need to go to the store!");
            }
        }
     }

Java recognized when a boolean is in the condition, so you don't have to do anything like groceries == true in the condition. However, you do need to do this if you are working with other data types.

The syntax is the same as an else if statement, but without the conditional part. Also, the keyword is just else. It looks like this:if(condition){code}else{code}
