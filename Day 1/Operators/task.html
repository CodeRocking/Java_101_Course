What Can You Do With Variables?

Learning to How to Use Operators

In this task, you will learn all of the basic java operators and how to use them.

  • Addition is done by using + operator. Test this out by defining 2 integers with random values, and printing your result. Your code should look something like this: 
               
int a = 8;
int b = 29;
System.out.println(a + b);
    You can also assign the result to a variable and print out that variable like this: 
               
int a = 4;
int b = 20;
int c = a + b;
System.out.println(c);
    You can also just print out the result like this: System.out.println(96 + 42);
  • Subtraction is done by using the - operator. Test this out by replacing the + operator with the - in the examples above
  • Multiplication is done by using the * operator. Test this out by replacing the + operator with the * in the examples above
  • Division is done by using the / operator. Test this out by replacing the + operator with the / in the examples above

Now that we know how to do basic arithmetic in java let's look at some of the special operators

  • The modulus (%) operation finds the remainder after a division of one number by another.
  • You can assign a value to variable using one equal sign =
  • You can check if to values/variables are equal by using two equal signs == This is a very important distinction which
